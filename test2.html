This is another test file. It can be used to document the feature that allows for multiple documents to be opened from the same instance of WebWhirr.

To use this, when asked to select an HTML file, select one of the test files included with WebWhirr--that is, test.html, test2.html, or test3.html.

Eventually, WebWhirr will gain more features, such as the ability to connect to the internet and a fully featured, customizable interface. For now, however, the focus is on building the rendering base that will be used in the browser.
